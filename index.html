通知设置

配置构建失败告警的通知渠道，均支持多收件人。在输入框中填写值，点击「复制并设置」按钮，将生成的命令在终端执行即可完成配置。
飞书/钉钉使用应用 API 发送个人消息；邮件使用 SMTP 发送。已配置的敏感值仅显示「已配置」。

飞书 (个人消息)

gh secret set FEISHU_APP_ID --repo winson-00178005/build-eye

钉钉 (个人消息)

gh secret set DINGTALK_APP_KEY --repo winson-00178005/build-eye

Email (邮件)

gh secret set SMTP_HOST --repo winson-00178005/build-eye
